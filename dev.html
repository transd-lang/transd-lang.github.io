2020-10-26 DONE Associative arrray ('Index') is added.

2020-10-17 DONE Exceptions are added.

2020-10-15 NEW Implement a new container type: associative array.

2020-10-13 NEW Implement break and continue in the while loop.

2020-10-13 NEW Implement exception handling mechanism.

2020-10-13 DONE More advanced example demonstrating key features of Transd "Demo_graph" is released.

2020-10-12 DONE The first C++ demo program "Demo_tiny" is released.

2020-10-11 DONE Transd shell "FREND" is released.

2020-10-07 NEW Create and upload to GitHub an example C++ program demonstrating using Transd for extending C++ applications.

2020-10-02 DONE A more advanced version of the "proof-of-concept" program is ready.

2020-09-28 PROGRESS Transd shell is almost ready

2020-09-27 NEW Improve the source files loading mechanism so that source files could be loaded from the shell and functions from them could be called.

2020-09-20 NEW Create a more advanced version of the "proof-of-concept" program where the capabilities of classes are demonstrated.
