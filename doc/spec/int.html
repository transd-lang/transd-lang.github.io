Int




Int - this type represents signed integer numbers. Corresponds to ‘long int’ in C++.

The size of Int is platform-dependent. On some systems (e.g. x64 Linux) Int is 8-byte long, on others (e.g. Windows, x86 Linux) Int is 4-bytes. Generally, it is guaranteed to be at least 32-bit, but if a Transd program is only intended to run on x64 Linux, then one can use the additional capacity of Int this platform offers.

4-byte Int has the range of values:

-2,147,483,648 ... 2,147,483,647

8-byte Int has the range of values:

-9,223,372,036,854,775,808 ... 9,223,372,036,854,775,807
