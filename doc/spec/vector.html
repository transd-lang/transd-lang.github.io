Vector



Vector - a container type representing an array of values of single type. Vectors in Transd are strongly typed: the type of vector's elements is specified at vector declaration and becomes an unchangeable part of vector's type. On this page the type of vector element is further referred to as ‘ElType’.

Elements are stored in an one-dimensional sequence and can be accessed by the number (index) of their place in the sequence.

Below is the list of the Vector class methods. ‘subj’ in the parameter list is the Vector object on which the method is called.

constructors



Vector(ElType)
ElType - a type specifier.

Creates an empty vector for storing values of type ElType.


Vector(ElType n Int())
ElType - a type specifier;
n - a non-negative integer.

Creates an empty vector for storing values of type ElType and fills it with n ElType elements initialized to the default ElType value.


[arg1, arg2, ...]

Creates an empty vector for storing values of the first argument's type and fills it with elements arg1, arg2, ... which all must belong to the same type.

Examples
v1: Vector( Int() ),
v2: Vector( Int() 10 ),
v3: [1, 2, 3, 4, 5]


add
(add subj el ElType)

Return type: ElType.

Appends the element el to the end of vector subj. The length of subj is increased by one.


get


(get subj idx Int())

Return type: ElType.

Returns the reference to the element with index idx. idx must refer to a valid position in subj (i.e. idx must be in the range from 0 to size - 1, where size is the size of subj).


set


(set subj idx Int() el ElType)

Return type: Null().

Assigns the element el to the position with index idx in vector subj. idx must refer to a valid position in subj.


size


(size subj)

Return type: Int().

Returns the size of subj.


resize


(resize subj num Int())

Return type: Null().

Sets the new size of subj equal to num. If num is less than the current size of subj, then excess elements are discarded from the vector's end. If num is greater than the current size of subj, then new elements with default values are added to the vector's end.


clear


(clear subj)

Return type: Null().

Discardes all elements of subj and sets the subj's size to 0.
