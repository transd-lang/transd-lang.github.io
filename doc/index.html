Documentation

Transd ecosystem currently comprises Transd language itself, FREND read-eval-print-loop program and Tourbillon virtual compiler.

The name Transd refers to the programming language as defined by the specification. This specification defines the syntax and semantics of Transd and can be realized by various implementations.

Tourbillon virtual compiler is an embeddable C++ implementation of Transd. Tourbillon has been created specially for Transd, but in theory can be used as a back-end for other programming languages, when provided with the compatible abstract syntax trees.

FREND is a REPL (read-eval-print-loop) program for evaluating at the command prompt Transd expressions, loading files and running Transd programs.
